inout is a simple, no-configuration file sharing tool. Install it on ONE device, tap start, and anyone on the same network can upload and download files instantly via browser — no app needed on the other side.

Just pick a folder, tap start, and share.

Features:
• One-tap file sharing via HTTP
• Upload files from any browser
• QR code for easy access
• File transfer log
• Customizable permissions (read-only, upload, full)
• Multi-language support (English, 简体中文, 繁體中文)
• Material 3 design with 6 color themes
• Desktop drag & drop support
• System tray integration

Your files stay on your local network — nothing leaves your device.
